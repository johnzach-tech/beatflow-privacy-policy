Privacy Policy – BeatFlow Music App

BeatFlow Music App menghargai dan melindungi privasi pengguna. Aplikasi ini dirancang untuk memberikan layanan pemutar musik dan tidak mengumpulkan data pribadi pengguna tanpa izin.

Informasi yang Diakses

Aplikasi BeatFlow dapat mengakses:

  • Penyimpanan perangkat untuk membaca file musik
  • Koneksi internet untuk fitur online

Aplikasi tidak mengumpulkan informasi sensitif seperti kata sandi, data perbankan, atau identitas pribadi pengguna.

Penggunaan Data

Data yang diakses hanya digunakan untuk:

  • Menampilkan dan memutar file musik
  • Meningkatkan pengalaman pengguna
  • Mendukung fungsi aplikasi

Keamanan Data

Kami menjaga keamanan data pengguna dan tidak membagikan informasi pengguna kepada pihak ketiga.

Perubahan Kebijakan

Kebijakan privasi ini dapat diperbarui sewaktu-waktu sesuai kebutuhan pengembangan aplikasi.
